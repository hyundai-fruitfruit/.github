### 기획 배경
- 더현대 서울 온라인 어플 기획에 앞서 더현대 서울 오프라인 공간 기획에 주목
- 오프라인 공간 경험을 극대화하고 더현대 서울의 브랜드 아이덴티티를 강화하도록 기획

![image](https://github.com/hyundai-fruitfruit/.github/assets/63828057/0679c481-53d3-46ce-97c0-28ca4f04d584)

### 주요 기능
- 3D 캐릭터 및 방 꾸미기
- 랜덤 스팟 이벤트
- 챗봇 흰디
